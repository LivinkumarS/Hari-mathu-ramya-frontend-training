DOM (Document Object Model)
----

the DOM is a tree-like structure representing the HTML elements on a webpage.

html ---> head ---> meta, meta, title
          body --->div.parent---->div.child-->h1--->span.
                   script         div.child-->h1--->span.

each node is an object---> all the details about that element.

Selecting Elements in DOM 
--------------------------

getElementById("id name")---> selecting an element by it's Id name.

getElementsByClassName("class name")---> selecting an element by it's class name.

getElementsByTagName("tag name")---> selecting an element by it's tag name.

querySelector("query") ---> returns only one element.
querySelectorAll("query") ---> returns all the elements.


Manipulating Contents.
-----------------------
textContent---> used to manipulate or access the text inside an element.
innerHTML ---> used to manipulate or access the HTML structure in an element.

Manipulating Attributes 
-------------------------
getAttribute("name")---> used to access the attribule inside an element.
setAttribute("name","value");
removeAttribute("name")

Manipulating Styles
----------------------
element.style.propertyName="value"
propertyName---> camelCase

ClassName Manipulation
-----------------------
element.classList----> accessing class values
classList.add("value")--> adding class name
classList.remove("value")--> removing class name
classList.toggle("value")--> toggling class name

DOM Tree navigation
-------------------
.children---> returns all the child elements as an array.
.firstElementChild---> returns the  first child element
.lastElementChild---> returns the  first child element


.parentElement---> accessing parent element

.previousElementSibling,.nextElementSibling---> returns sibling element.

Creating Element 
--------------------
createElement("tag name")---> to create an element.

Inserting
--------

appendChild---> adding an element at the end.
prepand----> at the beginning.

insertBefore(element, ref)
